# Blockspace
Blockspace is a small game coded in java using chatGPT codex, you can place moving boxes, rigid boxes, light, erase boxes, and even fly.

Do download, download the latest release, extract it, and run launch.vbs. If you make any changes to the code, run gradlew.bat first to compile it. You need to install java 21 and gradlew for it to run.
